How we work

We use rapid prototyping

What you won’t find us doing is spending weeks perfecting high quality design mocks. Making such serious investment and commitment to a design so early on is a flawed way to work.

Instead, expect from us enough visual documentation to help convey an idea. Once we all feel like it’s the right choice. We start bringing the idea to life with code.

How we design at DCUBED

We ship fast, and ship often.

Using platforms like Heroku, we are able to create and host any kind of app in seconds.

Not only does this automated deployment process save time (money), it means you get a publically accessible link to your application upon kick off. You can now test and share with your own team, whenever we push a release.
